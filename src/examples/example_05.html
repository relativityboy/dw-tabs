Here we introduce event binding with the options object

Choose a tab and watch the name appear in the disabled input element above. You can write your own function to do animations, or make other interesting things happen when a user clicks on a tab.

Bonus: Notice .showFirst is also set. This is an easy way of selecting which tab to show first, and can save you from having to set the 'active' class on your panels and tabs in your html to choose which tab to show 'on load'.
