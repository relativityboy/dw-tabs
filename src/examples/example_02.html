In this example we use all DW-Tabs defaults, but we use Bootstrap HTML & CSS

Note, as in Example 1....

All you need (if you're using the defaults) is to

  1. add the 'w-dwtab' class and the 'data-dwtarget' attribute to your tab elements. i
  2. add the 'w-dwcontent' class and the 'data-dwtarget' attribute to your content elements i
  3. add the 'active' class to the tab/content elements you want to have show up first. ii
  4. call $('.w-navbar').dwTabs('.w-navcontent'); like in the example above

  • the value of 'data-dwtarget' match in the tab/content elements needs to match for corresponding tabs/content
  • If you haven't already, add some css like in the head of this document to show+highlight the content+tab that are 'active'
  • It is not recommended that you apply css styling to classes beginning with 'w-'. 'w-' classes are reserved for web-application behavior attachment.
