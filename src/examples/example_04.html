A simple example to show you how to switch tabs in Javascript.

Choose a tab name from the dropdown above to switch to it. Obviously the tabs themselves update, and work as expected. You can see from the code below, we get the dwTabs instance directly from the dom (though you could easily cache it in a variable if you wanted).

Bonus: There are 2 extra items in the select list, one is 'headless' content (doesn't have a tab), and the other will show you what happens when you pass in an invalid tab name.
